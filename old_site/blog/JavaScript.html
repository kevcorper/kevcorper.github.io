Javascript vs. Ruby

March 3, 2016

Javascript Loops
Loops in Javascript have much more limited options than Ruby. In Javascript, the looping devices that exist are while and for loops. While loops use a condition that must be true to continue looping through a block of code. For loops specify an index counter (which is a common need in programming), a conditional, and a counter manipulator, which is often used to iterate through arrays and objects. Examples of these loops can be seen below.

While Loop Example

var x = true

while (x) {
console.log("This will print once")
x = false
};

=> "This will print once"

For Loop Example

for (i = 0; i < 5; i++)
console.log(i);

=> 0
=> 1
=> 2
=> 3
=> 4

In Comparison to Ruby
Ruby offers plenty more looping capabilities than Javascript in terms of built-in methods. The most notable of these being the each method that iterates through arrays and hashes, and the collection of enumerables that come along with it. Javascript has similar enumerable-like methods for some of the objects (such as map), but it does rely heavier on more from-scratch methods than Ruby.
