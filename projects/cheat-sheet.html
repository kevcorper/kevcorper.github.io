Ruby: A Basic Overview

What is Ruby?

Ruby is a dynamic, object-oriented programming language developed in the 90s and has been gaining popularity since. The language utilizes a more simplistic and legible structure to its code and contains an enormous library of methods. This leads to the growing number of Ruby users to consider it a language that is artful and simple yet incredibly complex under the hood. The flexibility of the language with libraries (gems) and frameworks such as the extremely popular Ruby on Rails.

Data Types

Everything in Ruby is an object, which allows for every bit of information to be given their own properties and methods. This means that even an integer has inherent methods that can built upon.

Strings: Created using the String::new object creator or enclosed in single or double quotation marks as literals, strings are characters.

Numbers: Integers and floats are considered numbers when written as is, with integers containing whole numbers and floats decimals.

Symbols: Although creating everything as an object is great for Ruby's flexibility, it also adds a cost of slower code due to the fact that Ruby is creating every bit of information as an object (even identical strings). Symbols, created using a colon, allow for Ruby to use a lighter object that allows for faster code because they create unique objects that will always have the same objectID.

Arrays: Arrays are lists, specifically lists that rely on order. They are created inside brackets and use an index notation to call upon items in the list. This index notation begins at 0 and can be called inside brackets to select certain items (array[index]).

Hashes: Hashes are created in curly brackets or literally with a hash constructor. They are also lists, but contain a bit more information where order is not as important as meaning and value. Hashes use key pairs (a key and value) to store information, where values can be selected by calling the key, which are often symbols, on the hash (hash[:key]).

Variables: Variables are created on the spot without needing explicit declaration. Setting any value beginning with a letter equal to any value will automatically create a variable. Depending on the location of variable instantiation, they can be local(using lowercase camelCase), instance(using an @ symbol), constant(all caps) or global(using $).

Methods: Ruby methods are very similar to functions in other langauges. They are created using the def keyword and utilize a block of code to be called upon any objects as long a the arguments match up.

Loops

Ruby loops are used to execute the same block of code for an x amount of times.

While: While loops execute the code while a condition is true.

Until: Until loops execute the code until a condition is true.

For-in: For-in loops execute the code in a certain expression for as large as the expression is (used on arrays and hashes). The for-in loop is so similar to the Ruby each method that it often gets replaced with each.

Ruby utilizes statements within its loops as well for easy use. The break statement terminates the most inner loop you are currently in and the next statement allows for the loop to skip the current iteration and move on to the next without breaking.

Enumerables

Ruby enumerables utilize the method each, which iterates through an array or hash to execute a block of code for each element, to create further specific iterating methods on array and hashes. There is an extensive list of enumerables which can be found HERE, but I have included some of the enumerables that I have personally found most useful.

map/collect: returns a new array with the block of code affecting each element.

include?(): returns true or false if the array includes the specified value.

max/min: returns the maximum or minimum value of the array.

select: returns a new array with only the elements that satisfy a specific condition

map/collect: returns a new array with the block of code affecting each element.
