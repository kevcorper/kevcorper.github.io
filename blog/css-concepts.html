CSS: The Box Model

February 4, 2016

Padding
Padding is the area between content and its border. Padding can be used to either create a large background behind a certain element or can be set to 0 to create a border that is flesh with the content edge.

Border
The border is exactly what it sounds like--the border surrounds the content. It is important to note however that the border includes the padding of the content as well.

Margin
Margin is the area surround the content and the border, the outermost property of the box model. Margin can often be most usefully utilized when trying to move and reposition content.
